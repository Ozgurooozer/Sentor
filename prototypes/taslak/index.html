Kurulum Kılavuzu

Sentor'u sisteminize kurmak için adım adım talimatlar.

Gereksinimler

Python 3.10 veya üzeri. Sistemde pip ve git kurulu olmalıdır. Herhangi bir sanal ortam veya bağımlılık yöneticisi gerekmez — tüm araçlar Python standart kütüphanesi ile çalışır.

Kurulum

Depoyu klonlayın ve doğrudan kullanmaya başlayın:

git clone https://github.com/Ozgurooozer/Sentor
cd sentor
python main.py serve

Bu komut REST API sunucusunu başlatır. Varsayılan port 4242'dir.

Doğrulama

Tarayıcınızda http://localhost:4242/api/pages adresine giderek kurulumu doğrulayabilirsiniz.
