01 · Konsept

Orkestra Agent Nedir?

Tıpkı bir orkestra şefinin enstrümanları yönettiği gibi, Otlas uzman AI ajanlarını koordine ederek karmaşık işleri başarır.

♫

Şef (Conductor)

Görevi analiz eder, plan oluşturur, her ajana doğru talimatı verir ve sonuçları birleştirir. Orkestranın beynidir.

  • Görevi alt görevlere böler
  • Doğru ajanı seçer ve yönlendirir
  • İlerlemeyi izler ve hataları yönetir
  • Nihai çıktıyı sentezler
🎶

Uzman Ajanlar (Section)

Her biri tek bir konuda uzmanlaşmış bağımsız AI ajanları. Kemancılar, üflemeliler, vurmalılar gibi…

  • Her ajan tek bir yetenekte uzman
  • Bağımsız çalışır, paralel işlem yapabilir
  • Standartlaştırılmış girdi/çıktı formatı
  • Yeni ajanlar eklenebilir, çıkarılabilir

03 · Orchestration Akışı

Bir Görev Nasıl Yönetilir?

Orchestra Agent bir görevi aldığında izlediği adımlar.

01

Görevi Analiz Et

Kullanıcı girdisini ayrıştırır, niyeti anlar, hangi uzmanlıkların gerektiğini belirler.

parse → intent → requirements
02

Plan Oluştur

Görevi alt görevlere böler, bağımlılıkları belirler ve bir execution DAG oluşturur.

decompose → dependencies → schedule
03

Uygun Ajanları Seç

Her alt görev için en uygun uzman ajanı atar. Paralel çalışabilecek görevleri eş zamanlı başlatır.

match → assign → dispatch
04

İzle ve Koordine Et

Ajanların ilerlemesini gerçek zamanlı izler, hataları yakalar ve gerektiğinde yeniden yönlendirir.

monitor → handle errors → re-route
05

Sonuçları Birleştir

Tüm ajan çıktılarını tutarlı bir bütün halinde sentezler, çakışmaları giderir ve final çıktıyı üretir.

merge → resolve conflicts → synthesize
06

Doğrula ve Teslim Et

Nihai çıktıyı kalite kontrolünden geçirir, gerekirse revizyon döngüsü başlatır ve kullanıcıya sunar.

validate → revise → deliver

05 · Kullanım Senaryoları

Sentor ile Neler Yapılabilir?

Orchestra Agent ile karmaşık görevleri tek cümleyle başlatın.

📖 Sentor

"Kapsamlı bir doküman hazırla"

Research Agent bilgi toplar, Write Agent sayfayı yazar, Review Agent kaliteyi denetler. Sonuç: yeni vault sayfası.
Research → Write → Review → Index
💻 Geliştirme

"API'e yeni endpoint ekle"

Code Agent server kodunu yazar, Review Agent denetler, Index Agent dokümantasyonu günceller.
Code → Review → Index
🔍 Araştırma

"Vault'ta bu temayı araştır"

Search Agent ilgili sayfaları bulur, Research Agent sentezler, Write Agent özet sayfası oluşturur.
Search → Research → Write
🔧 Bakım

"Backlink'leri güncelle"

Index Agent tüm sayfaları yeniden indeksler, Search Agent kırık bağlantıları tespit eder, Write Agent düzeltir.
Index → Search → Write
