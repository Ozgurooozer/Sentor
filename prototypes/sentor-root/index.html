HOW IT WORKS
[01]
Write HTML

Create knowledge pages as plain HTML files. Drop them anywhere inside your vault directory.

[02]
Run Sentor index

The indexer scans your vault and builds a local Fuse.js search index from all pages.

[03]
Search

Query via browser UI, terminal CLI, or the REST API for your local AI agents.

FEATURES
browser

Browser UI

Full-featured search interface that runs directly in your browser. Open any vault page from results — no server, no build step required.

terminal

Terminal CLI

Query your vault from the command line. Pipe output, grep results, compose with other tools — it's just a program.

api

Ollama Agent API

Local AI agents query your knowledge base via REST. Combine Ollama models with your vault for retrieval-augmented answers — offline.

offline

Fully Offline

No account. No cloud sync. No telemetry. Your knowledge never leaves your machine — available on a plane, in a bunker, always.

STACK
