# Atlas OS — Keyboard Shortcuts

## Global

| Shortcut | Action |
|---|---|
| `Ctrl+T` | New terminal tab |
| `Ctrl+W` | Close active tab |
| `Ctrl+1` … `Ctrl+9` | Switch to tab by index |
| `Ctrl+Shift+E` | Toggle file explorer |
| `Ctrl+Shift+G` | Open graph view |
| `Ctrl+Shift+A` | Open agent switcher |
| `Ctrl+Shift+F` | Search in explorer |
| `Ctrl+Alt+F` | Toggle focused overlay mode |
| `Ctrl+Alt+P` | Toggle click-through (focused mode only) |

## Editor

| Shortcut | Action |
|---|---|
| `Ctrl+S` | Save file |
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+/` | Toggle line comment |
| `Ctrl+D` | Select next occurrence |
| `Ctrl+G` | Go to line |
| `Escape` | Enable vim normal mode (when vim mode is on) |

## Terminal

| Shortcut | Action |
|---|---|
| `Ctrl+Shift+\` | Split pane |
| `Ctrl+Shift+W` | Close pane |

## AI Chat

| Shortcut | Action |
|---|---|
| `Enter` | Send message |
| `Shift+Enter` | New line |
| `↑` | Previous message (in empty input) |
| `Ctrl+L` | Clear chat |

## Customising Shortcuts

Open Settings → Keyboard to rebind any shortcut. Shortcuts are stored in `~/.atlas/keybindings.json`.
