Bölüm 01

Zarafet — Semantic HTML

Doğru etiketi kullanmak, hem tarayıcıya hem insana anlam taşır. Gereksiz div yığını zarafeti öldürür.

Kötü
<div class="header">
  <div class="nav">
    <div class="nav-item">Ana Sayfa</div>
    <div class="nav-item">Hakkında</div>
  </div>
</div>
<div class="main">
  <div class="article">
    <div class="title">Başlık</div>
  </div>
</div>
Anlamsız div'ler — ekran okuyucu kör, SEO zayıf.
İyi
<header>
  <nav>
    <a href="/">Ana Sayfa</a>
    <a href="/hakkinda">Hakkında</a>
  </nav>
</header>
<main>
  <article>
    <h1>Başlık</h1>
  </article>
</main>
Semantic etiketler — erişilebilir, SEO dostu, zarif.
Bölüm 02

Temizlik — Gereksiz Kod Yok

Temiz kod; inline style, gereksiz class ve tekrar eden niteliklerden arınmış koddur.

Kirli
<button
  style="color:white;background:blue;
  padding:10px 20px;border:none;
  border-radius:5px;cursor:pointer;"
  onclick="doSomething()"
  class="btn btn-blue btn-main
  btn-action btn-submit">
  Gönder
</button>
Inline style + class karmaşası + davranış HTML'de.
Temiz
<button class="btn-primary"
        type="submit">
  Gönder
</button>

/* CSS'de: */
.btn-primary {
  padding: .75rem 1.5rem;
  background: var(--accent);
  border-radius: 8px;
}
Tek class, stil CSS'de, davranış JS'de. Sorumluluk ayrımı.
Bölüm 03

Optimizasyon — Hız ve Verimlilik

Optimize HTML; gereksiz istek yapmaz, render'ı bloklamaz, kaynakları doğru yükler.

Yavaş
<head>
  <!-- render bloklayan script -->
  <script src="app.js"></script>
  <script src="utils.js"></script>
</head>
<body>
  <!-- optimize edilmemiş resim -->
  <img src="photo.png">
</body>
Script head'de → sayfa donuyor. Resimde boyut/format yok.
Hızlı
<head>
  <!-- preload kritik kaynak -->
  <link rel="preload" href="app.js"
        as="script">
</head>
<body>
  <img src="photo.webp"
       width="800" height="600"
       loading="lazy"
       alt="Açıklama">
  <script src="app.js" defer></script>
</body>
defer + lazy loading + webp + boyut → hızlı, CLS yok.
Özet

4 Temel İlke

Kaliteli HTML'in özü dört ilkeye sığar.

🏗️

Semantic Yapı

Her etiket anlamını taşısın. header, main, article, section — div değil.

✂️

Minimal Kod

♿

Erişilebilirlik

⚡

Performans

defer, async, lazy loading, webp — her byte'ın sebebi olsun.
