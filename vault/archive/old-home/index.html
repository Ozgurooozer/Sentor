How to use

Add a page

Create vault/{category}/{slug}/index.html, then re-index.

python tools/indexer.py

Search in browser

Open ui/index.html — no server needed, works on file://.

ui/index.html

Search in terminal

Full-text search with ANSI output and category filters.

atlas search "query"

Start the API

Serves the knowledge base over HTTP for AI agents and scripts.

atlas serve

Chat with AI

Ollama agent queries your vault and cites sources from it.

atlas chat
