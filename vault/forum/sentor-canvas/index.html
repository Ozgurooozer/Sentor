Canvas Sistemi

Sentor 26 Mayıs 2026 · güncellendi 29 Mayıs 2026 Phase L · v0.12

Sentor'un varsayılan çalışma modu. Sonsuz pan/zoom tuval üzerinde node'lar (paneller) birbirine wire'larla bağlanır — veri akışı, AI prompt'ları, terminal komutları hepsi bu sistemden geçer.

Panel Özellikleri

  • Pinned: pinned: true — viewport'a sabitlenir, canvas'ın dışında kalır. PinnedPanelsPortal ile DOM'un en üstüne render edilir.
  • Minimized: minimized: true — CanvasDock şeridine küçülür. Tıklayınca geri açılır.
  • Sub-canvas: viewport + children alanları — kendi koordinat sisteminde iç canvas barındırır.
  • Accent rengi: Her tip için nodeAccent.ts'te tanımlı border rengi. Header tipi meta.headerColor'u override olarak kullanır.

Canvas Run Engine

▶ Run butonu artık gerçek bir topological execution engine çalıştırıyor. Kahn's BFS sort ile wire bağımlılıkları çözülür, her panel sırasıyla çalıştırılır. Gate paneli kapandığında downstream zinciri atlanır. Ayrıntılar: Canvas Run Engine + Gate Panel →

Arkaplan

Ana canvas arkaplanı V3CanvasBgPanel (Three.js WebGL, perspektif grid + partiküller + UnrealBloom). CSS tabanlı eski V3CanvasBgAmbient kaldırıldı — zoom/konum bağımsız, her zaman tam ekran görünür.

Minimap

Sağ alt köşede her zaman görünür (V3MiniMap). Panel yokken ortada crosshair gösterir; panel eklenince küçük dikdörtgen harita aktif olur. Haritaya tıklayınca o bölgeye zıplar.

Orkestra AI

Canvas'ı doğal dille yöneten entegre AI. Ollama, LM Studio ve OpenCode Zen'e doğrudan streaming fetch atar — AI SDK kullanmaz, bu sayede herhangi bir OpenAI uyumlu endpoint ile çalışır.

Nasıl çalışır: LLM yanıtının içine gömülü JSON tool call'ları arar: {"tool":"add_node","type":"chat","title":"Asistan"} gibi. Bunları parse edip canvasStore'a uygular. Kullanıcı "bir terminal ve chat ekle, birbirine bağla" diyebilir.

Dev CORS: orkestraStore.devProxy() dev modda opencode.ai adresini /opencode-proxy üzerinden yönlendirir. Production Tauri build'de doğrudan gider.

Orkestra araçları

  • add_node / add — yeni panel ekle
  • remove — panel kaldır
  • connect / wire — iki paneli wire ile bağla
  • build — N node ekle + bağla tek komutta
  • create_note — yapışkan not oluştur
  • list_nodes / list — mevcut panelleri listele
  • var_set / var_get / var_list — değişken yönetimi
