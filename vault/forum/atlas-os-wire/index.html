Wire Sistemi

Atlas OS 26 Mayıs 2026 Canvas · Veri Akışı

Canvas panelleri arası veri akış sistemi. Her wire bir bağlantı türü taşır ve downstream panelin nasıl yorumlayacağını belirler.

Üç Wire Tipi

data — Mavi
Açık değer wire. Chat panelinde badge satırında gösterilir. Kullanıcı hangi veriyi aldığını görür.
Örnek: Terminal çıktısı → Chat "Çalışma dizini:" badge'i
context — Mor
Sessiz arka plan wire. Her prompt'un başına otomatik eklenir, kullanıcıya ayrıca gösterilmez. AI her zaman bu bilgiyi görür.
Örnek: Editör içeriği → Chat (her soru sordukta dosyayı "bilir")
trigger — Yeşil
Execution sinyali. Değer taşımaz, sadece "çalıştır" sinyali gönderir. Chat → Terminal: AI'nın ürettiği komutu terminalde çalıştırır.
Örnek: Chat → Terminal trigger wire → AI komutu otomatik çalıştırılır

charLimit — Token Koruma

Her wire'ın ayrı karakter limiti vardır. Varsayılan: 4000 karakter.

Neden gerekli: Uzun terminal scrollback'i veya büyük dosya içeriği chat'e aktarılırsa model context penceresi dolabilir. Per-wire limit ile gürültülü kaynakları kısıtlayabilirsin, diğerlerini tam bırakabilirsin. 
// Bağlantı tanımı
{
  fromPanel: "terminal-1",
  toPanel:   "chat-1",
  kind:      "data",
  charLimit: 500    // terminal'den sadece son 500 char
}

outputData vs snapshotData

Paneller veriyi meta.outputData'ya yazar. Downstream wire'lar buradan okur.

AlanAçıklama
meta.outputDataCanlı değer. Panel çalıştıkça güncellenir.
meta.snapshotDataDonmuş snapshot. Set edilince canlı değeri gizler. Downstream sabit bir görünüm alır.
Snapshot modu: Terminal sürekli yeni çıktı üretirken "şu anki çıktıyı dondur" dersen chat sabit bir bağlam alır. Snapshot temizlenince tekrar canlı veri akar.

useAllIncomingWireData hook'u

// Chat paneli bu hook ile tüm upstream'i toplar
const blocks = useAllIncomingWireData(panelId);

// Her block:
{
  panelId, panelTitle, panelType,
  connectionKind,  // "data" | "context" | "trigger"
  charLimit,       // wire'ın limiti
  data: { kind: "text"|"image"|"json", value: ... } | null
}

Trigger wire'lar bu hook'a dahil edilmez — sadece değer taşıyan wire'lar toplanır.
