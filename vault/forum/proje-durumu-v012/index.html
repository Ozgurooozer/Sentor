Phase J — Glass Panel Interiors

8 canvas panel içeriği V3 glass estetiğine geçirildi. Tüm paneller artık şeffaf arkaplan kullanıyor — V3CanvasNode glass kart arkaplanı sağlıyor. İnteraktif elemanlar: rgba(255,255,255,0.05) bg / rgba(255,255,255,0.07) border / #c8c8d0 metin.

  • NotePanel — amber-glass tint, sıcak metin rengi, köşe katlama
  • ChecklistPanel — custom SVG checkbox, progress bar (mavi → %100'de yeşil)
  • PipePanel — glass prompt textarea, durum dot glow
  • InputPanel, GalleryPanel, FileBrowserPanel, ToolPanel, HeaderPanel — glass tokens
shadcn/tailwind.css kaldırıldı: Phase I'da shadcn package.json'dan silinmişti ama CSS import globals.css'te kalmıştı — Vite 500 hatası veriyordu. Düzeltildi.

Phase K — Mimari Temizlik

MCP konsolidasyonu

mcp/server.py (HTTP variant) silindi. Tek MCP sunucu: tools/mcp_server.py (stdio, zero-dep). _enqueue() atomic write'a geçirildi (tempfile → os.replace).

Event standardizasyonu

Tüm cross-window iletişim atlas: prefix ile tek namespace'e taşındı:

  • Cross-window → Tauri emitTo / listen
  • Same-window → Zustand store action (CustomEvent yok)

Canvas AI tools genişlemesi

buildCanvasTools() 6 structured tool kazandı: canvas_add_node, canvas_remove_node, canvas_update_node, canvas_connect, canvas_clear, canvas_send_to_terminal. Tam ajanlar (Claude/GPT) artık AI SDK tool use ile canvas'ı doğrudan yönetebilir.

Phase L — Blueprint/Variable Node Sistemi

Variable Store

Global Zustand + LazyStore persist (atlas-variables.json). setVariable / getVariable / listVariables / hydrate. scheduleFlush() timeout içinde fresh state okur — race-condition güvenli.

Yeni panel tipleri

TipAçıklama
variableGlobal değişken — set wire'ı yazar, output wire'ı broadcast eder. meta.varName ile bağlanır.
if-elseSaf dataflow. condition wire → true/false branch. Falsy: boş, "false", "0", "null".
for-eachJSON array veya newline-text → items_json output wire.

Orkestra değişken tools

Küçük modeller için: var_set, var_get, var_list (JSON scanning). Tam ajanlar için: variable_set, variable_get, variable_list (structured tool use). buildSystem() mevcut değişken listesini system prompt'a ekler.
