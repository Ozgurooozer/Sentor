Phase G — Canvas Geçişinin Detayı

Canvas artık varsayılan çalışma modu (layoutMode: "canvas"). Eski classic/focused modlar silinmedi — preferences.ts'te default değiştirilerek geri dönülebilir.

  • 20 panel tipi: terminal, editor, chat, agent, canvas, pipeline, codegraph, input, note, sketch, tool, pipe, checklist, header, gallery, filebrowser, web, vault-home, preview, instance
  • Wire sistemi: data (mavi) / context (mor) / trigger (yeşil). Per-wire charLimit, snapshotData
  • Orkestra AI: Doğal dille canvas kontrolü. LLM yanıtından JSON tool call'ları parse eder
  • TweaksPanel: Arka plan stili, wire animasyonu, kılavuz çizgiler, yoğunluk ayarları
  • Multi-canvas: CanvasRecord, canvasHistory, per-canvas JSON persist
  • Portlar: portDefs.ts'te her panel tipi için isimlendirilmiş typed port'lar
