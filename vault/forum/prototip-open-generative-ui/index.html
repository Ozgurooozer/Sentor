Prototip: OpenGenerativeUI — Atlas OS Entegrasyonu

Prototip 29 Mayıs 2026 Geliştiriliyor

OpenGenerativeUI (CopilotKit/OpenGenerativeUI) — AI'nın sadece metin değil, interaktif görsel bileşenler (HTML/SVG/Chart.js/Three.js) üretmesini sağlayan bir showcase projesi. CopilotKit v2 + LangChain Deep Agents ile çalışıyor.

Ne İşe Yarar

  • Algorithm visualizations — binary search, BFS vs DFS, sorting
  • 3D animations — interaktif WebGL/CSS3D sahneleri
  • Charts & diagrams — pie/bar charts, network diagrams (Recharts, Chart.js)
  • Interactive widgets — formlar, simülasyonlar, matematik plotları
  • Tüm görseller sandboxed iframe içinde render edilir, dark/light tema otomatik

Entegrasyon Planı

1. Atlas OS Modül Yapısı

modules/open-generative-ui/ altında bağımsız bir modül olarak çalışacak. Kendi package.json'ı ile bağımsız build alabilir, Atlas OS IDE bundler'ı ile dahil edilebilir.

2. Canvas Paneli

Yeni bir panel tipi (generative-ui) eklenerek CopilotKit'in useComponent hook'unu saran bir React komponenti yapılacak. Agent mesajlarındaki HTML/SVG çıktılarını sandboxed iframe'de gösterecek.

3. Agent Entegrasyonu

Atlas OS Canvas AI agent'ı (Orkestra), OpenGenerativeUI'nin LangGraph Deep Agent'ını bir alt-agent veya tool olarak çağırabilecek. Bu sayede kullanıcı "bana binary search görselleştirmesi yap" dediğinde canvas'ta bir Generative UI paneli açılacak.

4. MCP Server

OpenGenerativeUI'nin kendi MCP server'ı (apps/mcp) Atlas OS MCP tool'larına eklenecek. assemble_document ve skill resource'ları kullanılabilecek.

Teknik Zorluklar

  • Model gereksinimi: Generative UI güçlü modeller ister (GPT-5.4, Claude Opus 4, Gemini 3.1 Pro). Küçük modellerde çıktı kalitesi düşer.
  • Sandbox güvenliği: iframe içinde üretilen HTML'in güvenli şekilde render edilmesi (sandbox attribute, CSP).
  • State yönetimi: CopilotKit agent state pattern ile Atlas OS variable store arasında köprü.
  • Bağımlılık yükü: Next.js 16 + Turborepo + pnpm — Atlas OS'un zero-dep felsefesiyle çelişebilir.
