Self-Check Bulguları (Can + Sena)

⚡⚿ Şeytanın Avukatı

  • ThreadingHTTPServer geçişi tek başına yetersiz: _sem_records / _sem_mtime ve _Cache.pages() Lock'suz global state — geçiş yapılırsa race condition gizlenir değil, görünür hale gelir. Önce Lock, sonra geçiş.
  • CI'da cargo build tehlikeli: WebView2Loader.dll GitHub Actions runner'ında yok → runtime panic. Çözüm: cargo check + clippy yeter, release build tag push'a ertelenir.
  • Silent exception audit kapsamı dar: api/server.py düzeltildi ama tools/mcp_server.py ve cli/sentor.py içinde 35+ bare except: pass mevcut — dış servislerle konuşan kod sessiz hata yutmamalı.
  • Concurrent test sunucu sınıfıyla eşleşmeli: test ortamındaki server sınıfı üretimle aynı olmazsa concurrent test anlamsız.
