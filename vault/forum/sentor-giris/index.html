Sentor Nedir?

Sentor 26 Mayıs 2026 Genel Tanıtım

Tek Cümle

Sentor, tamamen offline çalışan, sıfır-bağımlılıklı bir kişisel bilgi tabanı + AI IDE + canvas iş akışı motorudur.

Felsefe: İnternet yok, bulut yok, abonelik yok. Her şey kendi makinende çalışır. Vault sayfaları düz HTML dosyalarıdır — tarayıcıda, terminalde, her yerde açılır.

Katmanlar

Kullanıcı
 │
 ├── IDE (Tauri v2 + React)          ← masaüstü uygulama
 │     ├── Canvas (20 panel, wire)   ← görsel iş akışı
 │     ├── AI Agents (3 agent)       ← Vault, Coder, Sentor-Maker
 │     ├── Terminal / Editor         ← geliştirme araçları
 │     └── MCP bridge                ← dış AI entegrasyonu
 │
 ├── CLI (Python stdlib)             ← terminal arayüzü
 │     ├── sentor search / chat
 │     ├── sentor pipeline run
 │     └── sentor serve-daemon
 │
 ├── REST API (port 4242)            ← programatik erişim
 │
 └── Vault                           ← bilgi tabanı
       vault/{kategori}/{slug}/index.html

Çalıştırmak

# IDE (önerilen)
sentor.bat

# Sadece API
python api/server.py

# CLI
python cli/main.py search "canvas wire"

# MCP sunucusu (dış AI araçları için)
python tools/mcp_server.py

Neden HTML Vault?

  • Sıfır bağımlılık — Python, pip, npm gerektirmez
  • Her yerde açılır — tarayıcı, terminal, IDE, iframe
  • Git dostu — her değişiklik diff olarak görünür
  • Agent dostu — AI araçları vault_search ile okur, vault_write ile yazar
  • Taşınabilir — USB'ye kopyala, offline çalışır
