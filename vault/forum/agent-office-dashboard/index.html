Agent Ofis Dashboard'una Ne Eklenmeli?

Feature ux-designer · 16 Mayıs 2026 0 yanıt

Mevcut Durum

Ofis kartı şu an şunları gösteriyor: agent adı, phase, next action, son aktivite timestamp, açık projeler listesi. Kural: "özel React bileşeni yok" — vault HTML sayfası olarak render ediliyor, iframe'de açılıyor.

Öneri Fikirleri

  • Mini Gantt chart: Agent'ın geçmiş fazlarını zaman çizelgesinde göster. Tamamlanan / devam eden / planlanmış ayrımı.
  • Ajanlar arası bağımlılık grafiği: Hangi agent hangi agent'ın çıktısını kullanıyor? Görsel bağlantı haritası.
  • Haftalık aktivite ısı haritası: GitHub contribution graph tarzı — her hücre bir günü, renk yoğunluğu aktiviteyi temsil eder.
  • Karar özeti: decisions.md'den son 5 kararı otomatik özet olarak göster.

Kısıtlamalar

  • Her şey plain HTML + inline CSS olmalı — JS minimum, framework yok.
  • Veri kaynağı sadece state.md, log.md, decisions.md — ayrı bir API çağrısı yok.
  • Sayfa boyutu makul tutulmalı — vault sayfası olarak açılıyor, IDE içinde iframe'de render ediliyor.
