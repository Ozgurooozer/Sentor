Built-in Agent'lar

Coder
builtin:coder
Genel kodlama. Yazar, düzenler, çalıştırır. En küçük doğru değişiklik.
Architect
builtin:architect
Tasarım ve trade-off analizi. Kod yazmadan önce plan. Risk ve ölçeklenebilirlik.
Code Reviewer
builtin:reviewer
Diff inceleme. Mantık hataları, perf, güvenlik. Linter'ın yakalayamadıkları.
Security
builtin:security
Tehdit modelleme. SSRF, path traversal, injection, CVE tespiti.
Designer
builtin:designer
UI/UX eleştiri. Hiyerarşi, boşluk, kontrast. Tailwind değerleriyle somut öneriler.
Debugger
builtin:debug
Hızlı hata tespiti. Hipotez → kök neden → minimal düzeltme → test.
Explain
builtin:explain
Kod açıklama. Sadece okur, hiçbir zaman düzenlemez. Genel bakış + kritik noktalar.
Test Writer
builtin:tests
Unit ve entegrasyon testi. Framework'ü otomatik algılar. Arrange-act-assert.
Web
builtin:web
Her cevap vault'a HTML sayfa olarak kaydedilir. Hesap makinaları, araçlar, makaleler.
Vault
builtin:vault
Hafıza döngüsü. Her turda vault_search → vault_read → cevap → vault_write.

GET /api/categories—Kategori listesi + sayfa sayısı
