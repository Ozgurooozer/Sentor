HTML İleri Teknikleri

Bu sayfada modern web geliştirmede kullanılan çeşitli ileri HTML tekniklerini detaylı olarak inceliyoruz.

Web Components

Web Components, yeniden kullanılabilir ve kapsüllenmiş HTML elementleri oluşturmamızı sağlayan bir teknolojidir. Shadow DOM, Custom Elements ve HTML Templates'i içerir.

Custom Elements

Kendi HTML elementlerinizi tanımlayabilirsiniz:

<custom-element></custom-element>

Shadow DOM

Element'in iç yapısını dışardan erişilemez hale getirir:

this.attachShadow({ mode: 'open' });

Service Workers

Service Workers, web uygulamasının arka planda çalışmasını sağlayan JavaScript dosyalarıdır. Offline destek, push notifications ve background sync sağlar.

Kullanım Alanları

  • Offline caching
  • Background sync
  • Push notifications
  • Network interception

Örnek Kod

navigator.serviceWorker.register('/sw.js')
  .then(registration => console.log('SW registered'));

Progressive Web Apps (PWA)

PWA'lar, native app benzeri deneyim sunan web uygulamalarıdır. Manifest dosyası ve Service Worker ile çalışır.

Manifest Özellikleri

  • App icons
  • Theme colors
  • Display mode (standalone, fullscreen)
  • Start URL

Örnek Manifest

{
  "name": "My App",
  "short_name": "App",
  "start_url": "/",
  "display": "standalone"
}

WebGL

WebGL, web tarayıcılarında 3D grafikler oluşturmak için kullanılan JavaScript API'sidir. OpenGL ES 2.0 tabanlıdır.

Temel Bileşenler

  • Vertex Shader
  • Fragment Shader
  • Buffers
  • Textures

Örnek Shader

attribute vec2 a_position;
void main() {
  gl_Position = vec4(a_position, 0.0, 1.0);
}

Demo: WebGL Küp

Microdata

Microdata, HTML içeriğine makine tarafından okunabilir metadata eklemek için kullanılan bir spesifikasyondur. Arama motorları için semantik bilgi sağlar.

Schema.org

En yaygın kullanılan vocabulary'dir:

<div itemscope itemtype="https://schema.org/Person">
  <span itemprop="name">John Doe</span>
</div>

Yararları

  • Rich snippets in search results
  • Better SEO
  • Structured data for crawlers

Constraint Validation API

HTML5 form validation için kullanılan native API. JavaScript olmadan client-side validation sağlar.

Validation Attributes

  • required - Zorunlu alan
  • minlength/maxlength - Uzunluk sınırlaması
  • pattern - Regex pattern
  • min/max - Sayısal sınırlar

JavaScript API

if (!form.checkValidity()) {
  form.reportValidity();
}

Demo: Validation Form

CORS (Cross-Origin Resource Sharing)

CORS, web sayfalarının farklı origin'lerden kaynak yüklemesine izin veren bir mekanizmadır.

CORS Headers

  • Access-Control-Allow-Origin
  • Access-Control-Allow-Methods
  • Access-Control-Allow-Headers

HTML'de Kullanım

<img src="https://example.com/image.jpg" crossorigin="anonymous">

Demo: CORS Image

Data Attributes

data-* attributes, HTML elementlerine özel data depolamak için kullanılır. JavaScript ile erişilebilir.

Kullanım

<div data-user-id="123" data-role="admin"></div>

JavaScript Erişim

element.dataset.userId // "123"
element.dataset.role // "admin"

Intersection Observer API

Elementlerin viewport ile kesişimini gözlemlemek için kullanılan API. Lazy loading ve scroll animasyonları için idealdir.

Temel Kullanım

const observer = new IntersectionObserver(callback, options);
observer.observe(element);

Options

  • root - Gözlem root'u
  • rootMargin - Margin
  • threshold - Kesişim oranı
